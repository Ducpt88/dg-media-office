Output 1: Map

  • Nap vao brain index.
  • Cap nhat map tung bai.
  • Cap nhat master map tong.
  • Mo tren mobile web.

Output 2: Telegram

  • Gui text doc ngay.
  • Gui file mindmap.
  • Gui wiki va full script.
  • Gui master map/wiki tong.
1. Cau hinh ban dau
  • Tao bot Telegram.
  • Dien TELEGRAM_BOT_TOKEN.
  • Dien TELEGRAM_CHAT_ID.
  • Dien web host, web port, va nguon can quet.
2. Setup khong can code
  • npm.cmd run commercial-setup
  • Nhap bot token, chat id, va source URL.
  • Hoac chay mot dong voi --token, --chat, --source.
  • Script tu tao cau hinh va chay audit.
3. Chay thu
  • npm.cmd run web
  • npm.cmd run brain-public
  • npm.cmd run brain-watch-once
  • npm.cmd run brain-audit
4. Khach kiem tra ket qua
  • Mo link map tren dien thoai.
  • Xem bai moi nhat tren map.
  • Doc text trong Telegram.
  • Tai file mindmap va wiki.
5. Tieu chuan dat
  • Audit pass tat ca dong.
  • Web map mo duoc tren mobile.
  • Telegram gui duoc text va file.
  • Khach co tai lieu huong dan.
