Hướng dẫn sử dụng AI Knowledge Brain Bot

Brain Bot giúp chuyển video, fanpage, kênh và file nguồn thành tài liệu học nhanh: transcript, tóm tắt, mindmap, wiki và bản TXT gửi Telegram.

Mục tiêu: khách không cần code vẫn có thể cung cấp Telegram bot, chat ID, nguồn quét và bật chạy liên tục.

Tính năng

  • Nhận link từ Telegram hoặc nhập thủ công.
  • Tạo map kiến thức xem trên điện thoại.
  • Gửi TXT đọc nhanh về Telegram.
  • Chạy theo lịch và quét liên tục.
  • Xóa file media nặng sau khi xử lý.

Cách dùng

1. Chuẩn bị Telegram bot
Nhập bot token và chat ID.
2. Thêm nguồn
Dán link kênh, page, video hoặc file nguồn.
3. Chạy audit
Kiểm tra bot, nguồn, map và quyền gửi Telegram.
4. Bật chạy liên tục
Bot tự quét và gửi kết quả theo chu kỳ.

Lưu và xóa dữ liệu

File audio/video/raw media chỉ giữ trong lúc xử lý. Khi đã tạo tài liệu và nạp vào map, bot có thể cleanup để tránh đầy máy và nặng Telegram.

Bàn giao

Gồm trang sản phẩm, trang setup, demo map, file TXT giới thiệu và backend API chạy bot thật.
